This application runs the p5.serialserver which enables connectivity between local serial devices and web applications via the p5.serialport p5.js library.

It is recommended that you select, open, and close serial ports via your p5 sketch rather than this application but the methods for doing so for simple debugging are available here as well. Please note that if you "Open" or "Close" here it will also effect your web applications (opening/closing here will also open/close it for your web applications).
